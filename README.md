REQUIREMENTS

  • Windows 10 or 11 (Recommended) for the Companion app, 
    Director console, and streaming software (OBS/vMix).
  • Android Phone or Tablet (Recommended) for Director (Switcher).
  • Android Phone (LensHub APK) or iOS Phone (PWA) for 
    Camera / Director.

═══════════════════════════════════════════════════════════════
USING THE PWA (iOS / Browser)


  For both SWITCHER and CAMERA roles on iOS or desktop browsers:

  1. Open Safari (iOS) or Chrome (Desktop) and enter your server IP:
     Example: https://192.168.1.50:3001

  2. IMPORTANT: Tap "Advanced" → "Proceed" to bypass the 
     security warning (normal for local network apps).

  3. Tap the "Share" button (iOS) or browser menu (Android/PC) 
     and select "Add to Home Screen".

  4. Open the app from your home screen. It will now run in 
     full-screen mode like a native app.

  5. Select your role (Camera or Switcher)


═══════════════════════════════════════════════════════════════
  USING THE APK (Android Native)
═══════════════════════════════════════════════════════════════

  For both SWITCHER and CAMERA roles on Android:

  1. Install the LensHub.apk on your device.
  2. Open the app and enter the Server IP address shown on the
     LensHub Studio (PC) screen. (On the Terminal e.g. 192.168.1.50:3001)
  3. Select your role (Camera or Director)

═══════════════════════════════════════════════════════════════
  USING THE COMPANION (OBS Output)
═══════════════════════════════════════════════════════════════

  1. Double-click Start_Companion.bat on your PC.
  2. The Director console will open. 
  3. Select Program output in the Director console. 
  4. When cameras connect,
     a feed will pop out automatically after you set a camera to a program (tap a cam feed).
  5. It automatically switch to the next camera feed when you switch to the next camera in the Director/Switcher console. (this is the main feature).  
  6. In OBS, add a "Window Capture" source and select the 
     "LensHub Output" window for the desired camera.

═══════════════════════════════════════════════════════════════
  SUMMARY OF SETUP
═══════════════════════════════════════════════════════════════

  1. Start the Server: Double-click Start_Companion.bat.
  2. Connect Devices: Use IP or QR code on phones/tablets.
  3. Stream: Capture the pop-out windows in OBS.
